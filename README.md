## 💡프로젝트 소개

기억에 남는 미디어들을 잊지 않기 위한 저장소

### 프로젝트 목적

#### 1) DataBase

저장하고자 하는 미디어들을 Oracle을 이용하여 데이터베이스의 CRUD 기능을 구현한다.
  
#### 2) User Interface

웹에서의 사용자 편의성을 고려하여 예외처리 및 인터페이스 간단화
